# MeshKit 使用说明

本文档适合第一次使用 MeshKit 的人快速上手。

## 1. 启动方式

开发环境：

```bash
pnpm install
pnpm --filter core build
pnpm dev
```

分别启动：

```bash
pnpm dev:web
pnpm dev:desktop
pnpm dev:signaling
```

Desktop 安装包：

```bash
pnpm --filter desktop release:win
```

## 2. 文件传输

### 点对点发送

1. 打开“文件传输”。
2. 选择“点对点传输”。
3. 选择或拖入文件。
4. 选择目标设备。
5. 点击发送。
6. 等待对方接收并标记完成。

### 点对点接收

1. 打开“文件传输”，保持页面在线。
2. 收到提醒后点击接收。
3. 多文件时选择需要接收的文件。
4. 保存文件。
5. 全部保存后点击“标记为已完成”。

### 取件码发送

1. 打开“文件传输”。
2. 选择“生成取件码”。
3. 选择文件。
4. 将取件码、链接或二维码发给对方。
5. 保持页面开启，直到接收方完成。

### 取件码接收

1. 选择“输入取件码”，或直接打开邀请链接。
2. 输入取件码和密码（如果有）。
3. 选择文件保存。
4. 保存完成后点击“标记为已完成”。

## 3. 便签墙

1. 打开“便签墙”。
2. 输入昵称和房间号，或使用最近房间。
3. 可选：开启密码和加密配置。
4. 进入房间后添加、编辑、移动便签。
5. 用分享按钮生成链接或二维码邀请其他成员。

房间规则：

- 只有房主可以销毁房间。
- 房间销毁后其他成员会收到提醒。
- 移动端拖动画布时建议长按后再移动。

## 4. 加密聊天

1. 打开“加密聊天”。
2. 输入昵称和房间号。
3. 可选：设置房间密码。
4. 进入房间后发送消息。
5. 用分享按钮邀请其他成员。

房间规则：

- 只有房主可以销毁房间。
- 房间销毁后其他成员会退出当前房间。
- 邀请链接只建议发送给可信的人。

## 5. Desktop 共享中心

Desktop 的设置页可以开启共享服务。

典型流程：

1. 打开 Desktop。
2. 进入设置页，开启共享中心。
3. 复制邀请链接或展示二维码。
4. 手机或其他电脑打开链接即可进入 Web 分享页。

如果别人发来 MeshKit 邀请链接，可以在 Desktop 设置页导入。导入后会自动写入连接配置，并跳转到对应功能。

## 6. 常见问题

### 看不到其他设备

- 确认双方在同一局域网。
- 确认 signaling 或 Desktop 共享服务已启动。
- 检查防火墙是否允许 `3000`、`7000`、`8000` 端口。
- 尝试点击刷新设备或刷新 RTC。

### 邀请链接打不开

- 确认链接里的主机 IP 是对方设备能访问的局域网地址。
- Desktop 分享时确认共享中心处于运行状态。
- 手机访问时确认手机和电脑在同一个 Wi-Fi。

### 传输中断

- 传输期间不要关闭页面或切换功能。
- 尝试刷新 RTC 后重新发起。
- 大文件传输时保持屏幕常亮和网络稳定。
